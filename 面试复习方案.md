# 前端开发面试复习方案

## 📋 复习目标
基于简历内容，准备前端开发岗位技术面试，重点掌握 Vue.js、React.js、Node.js 及相关技术栈。

---

## 🎯 复习计划（建议 2-3 周）

### 第一周：前端基础 + 框架深入

#### Day 1-2: JavaScript 基础
**核心知识点：**
- [ ] 原型链与继承
- [ ] 闭包与作用域
- [ ] 异步编程（Promise, async/await, Event Loop）
- [ ] ES6+ 新特性（let/const, 箭头函数, 解构, 模块化）
- [ ] this 指向问题
- [ ] 深浅拷贝

**高频面试题：**
1. 手写 Promise（Promise.all, Promise.race）
2. 手写防抖节流
3. 手写深拷贝
4. 实现 call/apply/bind
5. Event Loop 执行顺序

---

#### Day 3-4: Vue.js 生态系统
**核心知识点：**
- [ ] Vue2 vs Vue3 核心区别（响应式原理变化）
- [ ] Vue 生命周期（父子组件执行顺序）
- [ ] 组件通信方式（props/$emit, provide/inject, eventBus, Vuex）
- [ ] computed vs watch
- [ ] v-model 原理及自定义
- [ ] Vue Router 路由守卫
- [ ] **Vuex 核心概念**（state, getters, mutations, actions, modules）

**简历相关重点：**
- [ ] **vue-i18n 国际化实现**（哈萨克斯坦项目）
  - 如何配置多语言
  - 如何动态切换语言
  - 如何在模板和 JS 中使用
- [ ] **ECharts 数据可视化**
  - 图表配置项
  - 响应式处理
  - 性能优化（按需加载）
- [ ] **Element UI + Vxe-Table**
  - 二次封装技巧
  - 表格性能优化

**高频面试题：**
1. Vue 响应式原理（Object.defineProperty vs Proxy）
2. Vue 的 diff 算法
3. computed 和 watch 的区别
4. Vue 组件生命周期（父子组件执行顺序）
5. keep-alive 原理
6. nextTick 原理

#### Day 5-6: React 生态系统
**核心知识点：**
- [ ] React 核心概念（Virtual DOM, JSX, 组件化）

- [ ] React Context API 使用场景
- [ ] **React 19 新特性**

- [ ] **Redux 核心原理**（store, reducer, middleware）

**简历相关重点：**
- [ ] **React Context API 状态管理**（仿亚马逊项目）
  - 何时使用 Context
  - 性能优化策略
  - 与 Redux 对比

**高频面试题：**
1. Virtual DOM 和 diff 算法
2. React 生命周期（Hooks 如何替代）
3. useEffect 的依赖数组如何工作
4. React 性能优化（memo, useMemo, useCallback）
5. Context 的局限性及优化
6. Redux 工作流程及中间件原理
